## Installation

#### CocoaPods
You can use [CocoaPods](http://cocoapods.org/) to install `YourLibrary` by adding it to your `Podfile`:

```ruby
# platform :ios, '9.0'

target 'PlexPilot' do
  # Comment the next line if you don't want to use dynamic frameworks
  use_frameworks!

  # Pods for PlexPilot
  pod 'GoogleMaps'
  pod 'GooglePlaces'
  pod 'ReachabilitySwift'

end
